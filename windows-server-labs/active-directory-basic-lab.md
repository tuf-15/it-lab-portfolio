# Active Directory Basic Lab

## Goal

Set up a basic Windows Server Active Directory lab environment.

## Lab environment

- Windows Server 2019
- Domain Controller: DC01
- Client PC: Windows 10 / Windows 11
- Virtualization: Proxmox / VirtualBox / VMware

## What I configured

- Installed Windows Server
- Added Active Directory Domain Services role
- Promoted server to Domain Controller
- Created a test domain
- Created test users
- Joined a client computer to the domain

## Screenshots

Screenshots will be added here:

## What I learned

In this lab I learned how Active Directory works at a basic level, how a domain controller is created, and how client computers can join a Windows domain.
